# IPv6Gen

Professional Vanity IPv6 Address Generator. Supports Hexspeak, Creative mappings, and RFC-compliant configuration snippets.

## Hosting
This project is hosted on GitHub Pages: [https://gerhardthomas.github.io/ipv6gen/](https://gerhardthomas.github.io/ipv6gen/)

## Features
- **Strict Hexspeak:** standard a-f, 0-9 mappings.
- **Creative Mappings:** extended mappings for all characters.
- **ASCII Bytes:** converts text to its hex representation.
- **RFC Compliant:** outputs canonical addresses.
- **Cross-Platform Snippets:** copy ready-to-use commands for Linux, Windows, and macOS.

## Local Development
Open `index.html` in any modern web browser.
